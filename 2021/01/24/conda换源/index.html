1.问题描述

最近在使用Deepin，为了方便管理python版本安装了Anaconda，无奈conda默认源服务器在国外，下载网速太慢，因此决定切换国内源提高速度，下面记录一下Linux系统下的换源方法

2.命令

  • 添加清华源

  • 设置搜索时显示通道地址

    conda config --set show_channel_urls yes

  • 查看当前的镜像源

    conda config --show

  • 查看添加的镜像

    conda config --get channels

  • 删除源

    conda config --remove-key channels
