1.const的含义

const 是 constant 的缩写，本意是不变的，不易改变的意思。在 C++ 中是用来修饰内置类型变量，自定义对象，成员函数，返回值，函数参数。C++ const 允许指定一个语义约束，编译器会强制实施这个约束，允许程序员告诉编译器某值是保持不变的。如果在编程中确实有某个值保持不变，就应该明确使用const，这样可以获得编译器的帮助。

2.const的作用

  • 定义常量

    const int a = 100; //定义一个类型为int的常量，并初始化为100

    const定义的变量只有类型为整数或枚举，且以常量初始化时才能作为常量表达式，其他时候只是一个限定的变量。

  • 类型检查

    const常量具有具体类型，编译器可以对其做安全检查；

    #define宏定义是没有数据类型的，只是简单的字符串替换，不能进行安全检查
